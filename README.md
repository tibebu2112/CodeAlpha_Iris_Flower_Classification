# CodeAlpha: Iris Flower Classification

## 📌 Project Overview
This project was developed during my **Data Science Internship at CodeAlpha**. The primary objective is to build a machine learning model capable of classifying Iris flowers into three distinct species (**Setosa, Versicolor, and Virginica**) based on their physical measurements.

## 📊 Dataset Information
The project utilizes the classic **Iris Dataset**, a staple in the data science community. It includes 150 instances with the following four features:
* Sepal Length (cm)
* Sepal Width (cm)
* Petal Length (cm)
* Petal Width (cm)

## 🛠️ Technologies & Libraries
* **Python** - Core programming language.
* **Pandas** - For data manipulation and analysis.
* **Scikit-Learn** - For model building (Random Forest) and data preprocessing.
* **Seaborn & Matplotlib** - For generating the Confusion Matrix and Feature Importance plots.

## 🚀 Machine Learning Workflow
1.  **Preprocessing:** Used `LabelEncoder` to convert categorical species names into numerical format.
2.  **Data Splitting:** Applied an 80/20 train-test split for unbiased performance measurement.
3.  **Model Selection:** Implemented a **Random Forest Classifier** with 100 estimators.
4.  **Evaluation:** Validated the model using a **Confusion Matrix** and **Classification Report** (Precision, Recall, F1-Score).



## 📈 Key Results
* **Model Accuracy:** 100% (on the test dataset).
* **Top Features:** Analysis showed that **Petal Length** and **Petal Width** are the most critical factors in identifying the species correctly.
